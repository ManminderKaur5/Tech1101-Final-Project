CSS Cheatsheet

CSS Properties

  • color - Defines the text color.
  • background-color - Defines the background color.
  • font-size - Defines the size of the font.
  • font-family - Defines the font type.
  • margin - Sets the outer margin of an element.
  • padding - Sets the inner padding of an element.
  • border - Defines the border of an element.
  • display - Defines the display style (block, inline, flex, etc.).
  • width, height - Defines the width and height of an element.
  • position - Defines the positioning method of an element (static, relative, absolute, fixed).
