JavaScript Cheatsheet

JavaScript Syntax

  • let, const, var - Declare variables.
  • function - Define a function.
  • document.getElementById() - Access an element by its ID.
  • event.preventDefault() - Prevent the default behavior of an event.
  • alert() - Display a pop-up alert box.
  • console.log() - Log messages to the console.
  • addEventListener() - Add an event listener to an element.
  • if...else - Conditional statements for decision making.
  • for, while - Loop through arrays or objects.
