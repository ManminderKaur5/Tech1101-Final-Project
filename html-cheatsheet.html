HTML Cheatsheet

Common HTML Tags

  • <html> - Defines an HTML document.
  • <head> - Contains metadata and link to external files.
  • <body> - Defines the body of the HTML document.
  • <div> - A container for grouping elements.
  • <h1>, <h2>, <h3>... - Header tags for titles.
  • <p> - Defines a paragraph.
  • <a> - Defines a hyperlink.
  • <img> - Embeds an image.
  • <form> - Defines a form.
  • <input> - Defines an input field.
